 Quando Usar?
Quando você desejar configurar alguma coisa para as Operações de Venda (Estimates, Sales Order, Invoices, etc.). Neste cadastro você pode habilitar e desabilitar uma série de campos e funcionalidades para cada tipo de operação.

 Para que serve a Sigla Inicial?
O conteúdo deste campo afeta diretamente a numeração da Operação. Você pode digitar uma sigla de até 3 letras que irá compor o número gerado para a operação. Como padrão a Sigla Inicial para Estimates é “E” e para Sales Order é “SO”. Invoices não possuem sigla inicial como padrão, mas nada impede que você defina.

 O que é o Tipo Padrão?
As operações possuem um comportamento característico. Por exemplo, o Tipo de Operação Estimates é diferente do Tipo de Operação Invoices. Tanto no comportamento como no objetivo. O que faz esta diferença para o sistema é o Tipo Padrão, ele é quem define o comportamento e funcionalidades do Tipo de Operação. Observe que quando você muda o tipo padrão, alguns campos ficam habilitados e desabilitados conforme o tipo escolhido, isto já é uma demonstração do que é característico para o tipo padrão selecionado.

 O Flag Não permite estoque negativo está marcado mas não funciona, por que?
Existem 4 (quatro) configurações neste cadastro que só funcionam se o usuário não tiver permissão para realizar a operação. As opções “ Respeita Limite de Compra do Cliente”, “ Não se o cliente estiver em atraso”, “ Não grava se o cliente estiver bloqueado” e “ Não permite estoque negativo” só são válidas para usuários que não possuem permissão para tais bloqueios. Em outras palavras, se o usuário está associado ao perfil Administrador, estes bloqueios não terão efeito pois ele pode realizar tudo dentro do sistema, contudo se o usuário estiver associado ao perfil Vendedor, estes bloqueios com certeza funcionarão.

 Para que serve o botão Copiar?
Este cadastro de tipos de operações possui muitos campos, e para cadastrar um novo tipo de operação você teria que informar todos eles. Como, normalmente, um novo tipo de operação deve ser semelhante a um outro que já está cadastrado, este botão Copiar serve para que você possa escolher este tipo de operação. Desta forma todos os valores dos campos do tipo de operação escolhida serão copiados para a nova operação que está sendo criada, bastando apenas você modificar o que deseja que seja diferente.

 Como funciona a pasta “Adicionais”?
Provavelmente todos as informações que você precisa já estão previstas pelo sistema, contudo se existir alguma que seja bem específica ao seu tipo de negócio, você pode adicionar nesta pasta. É uma forma de você criar novos campos ou colunas dinamicamente. Basta você marcar e preencher os títulos e pronto o novo campo ou coluna estará disponível para que você possa utilizá-lo na operação. Como a língua mãe é a inglesa, esta deve ser preenchida para que o novo campo possa aparecer no relatório da Operação.
