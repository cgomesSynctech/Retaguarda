 Quando Usar?
Sempre que você precisar pagar as contas que você já cadastrou no sistema, você deve utilizar esta tela. Nela você pode selecionar uma ou mais contas, mesmo que sejam de Fornecedores diferentes e depois dizer como irá pagar. Se o pagamento for com cheque, esta tela também imprimirá os cheques gerados, um para cada fornecedor.

 Como Utilizar a tela?
A primeira atitude é escolher as contas que você deseja pagar e selecionar. Se as Contas que você deseja pagar já estão aparecendo na lista, basta simplesmente selecionar as que serão pagas.  Caso contrário, você pode utilizar outros métodos de filtro.  O método de filtro mais simples é escolhendo o fornecedor, desta forma, apenas as contas do fornecedor escolhido aparecerão na lista. Clicando no botão Pesq. Avançada, outras opções de filtro de contas aparecem para que você consiga facilmente localizar todas que deseja pagar.
Depois de selecionadas as contas, você pode modificar o Memo para adicionar alguma informação que julgue necessária.  Marque agora as formas de pagamento que você irá utilizar para pagar as contas selecionadas, preencha o valor correto em cada forma de pagamento escolhida e agora é só salvar. As Contas que você marcou serão liquidadas e os recursos para pagá-las sairão da conta que você escolheu.

 Como faço para gerar um cheque para cada fornecedor?
Este já é o padrão utilizado pelo sistema quando você escolhe a forma de pagamento Cheque. Se existirem contas de vários fornecedores selecionadas, será gerado um cheque para cada Fornecedor.  Contudo se você precisar, por algum motivo, gerar novamente os cheques, com o botão direito do mouse você pode comandar esta operação.

 Posso pagar com cheques de clientes que estão na empresa?
O MaxShop só permite isto se você selecionar contas de apenas um Fornecedor. Você pode escolher quais cheques irá pagar ao fornecedor na Forma de Pagamento Cheques em Loja.

 Como informo os juros que paguei?
Na Coluna Juros. Nela você deve entrar com o valor de juros que você pagou pela conta.

 Como faço um Pagamento Parcial de uma Conta?
Quando você seleciona uma Conta para pagar, na coluna Pagamento aparece o valor que você irá pagar, que por padrão é o valor total dela. Contudo, se você não irá pagar toda a dívida, você pode modificar este valor de pagamento e o sistema considerará como um Pagamento Parcial.

 Como posso corrigir um Pagamento?
Todos os pagamentos ficam registrados no sistema e podem ser vistos no botão Histórico da própria tela. Neste histórico você pode pesquisar o pagamento que você está procurando e então poderá alterar ou até mesmo cancelar toda a operação. Observe que, dependendo do tipo de alteração que você for fazer, o fechamento do caixa, ou até mesmos os relatórios de extrato que você já tirou poderão não mais refletir a  nova realidade que você modificou.

 O sistema imprime Recibo para pagamentos?
O recibo será impresso se o Flag Imprimir Recibo estiver marcado. No MaxShop o Recibo é configurável, você pode editar seu texto, escolher os campos que deseja ver e sua formatação visual, para isto, basta escolher a opção Editar Recibo, disponível no botão direito do mouse para a tela.
