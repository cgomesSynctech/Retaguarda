 Quando Usar?
Este cadastro é normalmente utilizado para consultar e gerenciar as parcelas que estão em aberto ou ver como determinada parcela foi paga. Para cadastrar, normalmente ele só é utilizado para parcelas antigas que não foram geradas por vendas no sistema. As parcelas que são geradas pelas vendas possuem restrições para alterações neste cadastro, pois entendemos que elas pertencem à venda e devem ser alteradas nela.

 Como localizo as Parcelas a Receber de determinado cliente? Como localizo as Parcelas a Receber que a empresa tem?
No botão Ir Para existe esta consulta preparada. Se você deseja ver todas as parcelas da empresa você basta clicar nesta opção. Caso deseje ver apenas de determinado cliente, escolha o cliente primeiro e só depois clique na consulta do botão Ir Para. Todas estas consultas consideram o cliente quando ele está informado.

 Como posso ver Totais para as parcelas que pesquisei?
O Formato Tabela para este cadastro já está configurado para exibir estes totais. Se você desejar modificar ou ver outra coluna totalizada, basta ir às opções de Sumarização do próprio componente (com o botão direito do mouse). O tutorial de Interface Padrão exibe como este componente de tabela se comporta e os recursos que oferece.

 Pesquisei e encontrei a Parcela que procuro, por que não posso alterar seu valor?
Se o campo Valor e outros não estão permitindo alteração é porque esta parcela foi gerada a partir de uma venda. Neste caso a única forma de alterar a parcela é alterando a própria venda que a gerou.

 Para que serve o botão Reparcelar?
Este botão chama a tela de reparcelamento de dívida, já com a parcela selecionada. É apenas uma forma de agilizar o acesso a outras rotinas do sistema. Clique aqui para saber o que é o Reparcelamento.
