 Quando Usar?
Sempre que você quiser fazer uma venda a um cliente. A Invoice é a venda propriamente dita, e é a última da seqüência. Se sua empresa utiliza todo o processo, temos o primeiro contato com o cliente através da Estimate; quando o cliente aprova, ela é faturada em uma Sales Order, e quando todo o processo interno já está concluído a Sales Order é faturada em uma Invoice. Apenas a Invoice gera as contas a receber e toda a contabilidade envolvida numa venda.

 Como Utilizar a tela?
Para saber os procedimentos e o comportamento da tela, assista o tutorial que explica como funciona a Interface Padrão do MaxShop. Neste tutorial você aprenderá a incluir, pesquisar, alterar e apagar um registro, bem como o comportamento de alguns componentes mais utilizados. Como uma dica de preenchimento, aconselhamos que o cliente seja a primeira informação digitada, após isto você pode ir direto para informar os itens da operação e só depois você diz como serão os termos para pagamento, isto é, o plano de pagamento.

 Por que o nome do cliente ficou Vermelho?
Isto é uma indicação visual de que existe alguma coisa errada com ele. O nome do cliente fica vermelho sempre que ele possuir parcelas atrasadas ou cheques devolvidos. Neste caso, colocando o mouse sobre o nome do cliente você poderá ver o que é que está errado com ele. Clicando com o botão direito do mouse sobre ele você também tem acesso a outras informações e cadastros.

 Como faço uma Invoice a partir de uma Sales Order ou Estimate?
Isto é extremamente simples. Quando você informa o nome do cliente, o Maxshop verifica, automaticamente, se existem Estimates ou Sales Order abertas para ele. Caso existam, o Maxshop já pergunta se você deseja “aproveitar” estas operações. Ao responder SIM para a pergunta, uma tela se abre para que você possa escolher as operações e os itens de cada uma que farão parte da Invoice, esta flexibilidade existe porque consideramos que o cliente pode não querer todos os itens.
A outra opção para se chegar na mesma tela, é através do botão Estimates/S.O., que fica em vermelho, indicando que existem Estimates ou Sales Order em aberto para o cliente. Para saber como funciona esta tela de Aproveitamento de Operações veja a ajuda da própria tela.

A numeração da Invoice é automática ou manual?
Isto vai depender de como está configurada a operação, por padrão, a numeração é automática. No MaxShop temos uma tela exclusiva para a configuração das Operações, e Invoice é uma delas. É o cadastro de Tipos de Operações, nele você configura o comportamento e algumas funcionalidades da operação.

 Existem campos demais que não são necessários para mim, posso ocultá-los? Estou sentindo falta de algumas informações, existe como adicioná-las?
O MaxShop disponibiliza algumas informações que podem ser vistas ou não, conforme estiver configurado no cadastro de Tipos de Operações. Lá você diz o que deseja que apareça. Se você está sentindo falta de alguma informação você deve ativá-la neste cadastro; se você está vendo informações demais, desnecessariamente, lá também você pode ocultá-las.

 Posso copiar os Itens de outra Sales Order, Estimate ou mesmo Invoice?
Com o botão direito do mouse sobre os itens, você tem a opção Copiar Itens da Operação. Abre-se uma tela para que você localize a operação da qual você deseja copiar os seus itens. Isto agiliza muito quando o cliente compra normalmente os mesmos itens, você não precisaria digitá-los sempre.

 Como altero o Tax?
O Tax, normalmente, só precisa ser alterado quando você estiver fazendo uma venda para outro estado. Para estes casos ou outros mais específicos, você pode alterar o Tax no botão Outros, na opção Alterar Tax. Contudo, o outro tax já deve estar cadastrado para que você possa escolhê-lo. Para cadastrar um outro Tax você deve ir ao seu cadastro no menu Cadastros, Tabelas Auxiliares.

 Para que serve o botão Reembolsos?
É uma forma prática de você cobrar por algum serviço ou produtos que você comprou para o cliente. Suponhamos que sua empresa é uma Joalharia e aparece um namorado romântico que deseja dar uma jóia dentro de uma caixinha musical, contudo você não vende este tipo de caixa, mas para agradá-lo você se propõe comprá-la. Neste caso, ou outro mais comum ao seu tipo de negócio, você realizou uma compra de um produto que certamente será cobrado ao cliente. Assim, no momento em que você registrar a compra da caixinha, você pode vincular esta compra ao cliente e automaticamente o Maxshop considera esta compra como uma despesa reembolsável. Logo, assim que você informar o nome do cliente na Invoice, você verá em vermelho o botão reembolsos, indicando que existe algo a mais a cobrar do cliente.

 Como posso ver os pagamentos que o cliente realizou para uma Invoice?
No botão Outros, existe uma opção Ver Pagamentos, ela mostra todos os pagamentos que o cliente efetuou para a invoice que você pesquisou.

 Posso fazer pagamentos a partir do cadastro do Fornecedor?
Sim, é possível já chamar a tela de pagamentos. Isto com certeza agiliza os trabalhos rotineiros. No botão Outros, existe uma opção que se chama Pagar Contas. Clicando nela, você estará chamando automaticamente a tela de pagamentos com todas as contas em aberto para com o Fornecedor, faltando somente informar como irá pagar.
