 Quando Usar?
Sempre que você quiser receber pagamentos de clientes que mas que suas dívidas não estão cadastradas no sistema, ou quando o cliente estiver fazendo um Pré-Pagamento.

 Como Utilizar a tela?
O primeiro passo é informar o nome do cliente. Quando o cliente é escolhido, se ele possui algum cheque devolvido, lhe será exibido automaticamente, pois o sistema considera que este recebimento pode ser para resgatar um cheque desta natureza.
O próximo (e último) passo é informar como o cliente efetuando seu pagamento, marcando todas as formas de pagamento que ele estiver usando e em cada uma o respectivo valor e informações particulares.
Agora é só Salvar o Recebimento. Se o Flag de Pré-Pagamento estiver marcado o valor do recebimento será lançando como um crédito para a ficha do cliente, caso contrário, uma parcela a Receber será cadastrada e quitada com o recebimento; neste caso o sistema possibilitará informar uma conta de receita para o correto registro contábil da operação.

 Como posso corrigir um Recebimento?
Todos os recebimentos ficam registrados no sistema e podem ser vistos no botão Histórico da própria tela. Neste histórico você pode pesquisar o recebimento que você está procurando e então poderá alterar ou até mesmo cancelar todo o recebimento. Observe que, dependendo do tipo de alteração que você for fazer, o fechamento do caixa, ou até mesmos os relatórios de extrato que você já tirou poderão não mais refletir a  nova realidade que você modificou.

 O sistema imprime o Recibo para o Cliente?
O recibo será impresso se o Flag Imprimir Recibo estiver marcado. No MaxShop o Recibo é configurável, você pode editar seu texto, escolher os campos que deseja ver e sua formatação visual, para isto, basta escolher a opção Editar Recibo, disponível no botão direito do mouse para a tela.
