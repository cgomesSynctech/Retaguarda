 Quando Usar?
Sempre que você precisar fazer pagamentos de dívidas que não estão cadastradas no sistema e que você irá utilizar Cheque para pagar. Normalmente são dívidas corriqueiras como um lanche, garrafões de água, e outras dívidas de natureza aleatória. Se você estiver fazendo um pré-pagamento, com cheque, a um fornecedor, também deverá utilizar esta tela. Em resumo esta é uma tela de pagamento rápido, só que a única forma de pagamento e o cheque.

 Como Utilizar a tela?
O primeiro passo é informar o nome do fornecedor. Depois o valor do cheque. O memo servirá para você identificar melhor o motivo do pagamento.
Agora é só Salvar o pagamento. Se o Flag de Pré-Pagamento estiver marcado o valor do pagamento será lançando como um crédito para a empresa, caso contrário, uma Conta a Pagar será cadastrada e quitada com o pagamento; neste caso o sistema possibilitará informar uma conta de despesa para o correto registro contábil da operação. Se nenhuma conta for informada a conta Miscellaneous será a padrão.

 Como posso corrigir um Pagamento com Cheque?
Todos os pagamentos ficam registrados no sistema e podem ser vistos no botão Histórico da própria tela. Neste histórico você pode pesquisar o pagamento que você está procurando e então poderá alterar ou até mesmo cancelar toda a operação. Observe que, dependendo do tipo de alteração que você for fazer, os relatórios de extrato que você já tirou poderão não mais refletir a nova realidade que você modificou.

 O sistema imprime Recibo para o pagamento?
O recibo será impresso se o Flag Imprimir Recibo estiver marcado. No MaxShop o Recibo é configurável, você pode editar seu texto, escolher os campos que deseja ver e sua formatação visual, para isto, basta escolher a opção Editar Recibo, disponível no botão direito do mouse para a tela.
